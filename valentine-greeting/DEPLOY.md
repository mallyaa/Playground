# Get a shareable link (PC + mobile)

Your Valentine page is one file: `index.html`. To get a link you can send that works on any phone or computer, host it online using one of these free options.

---

## Option 1: Netlify (fastest – no account needed for first time)

1. Go to **https://app.netlify.com/drop**
2. Drag and drop your **Playground** folder (the folder that contains `index.html`) onto the page.
3. Netlify will give you a link like `https://random-name-123.netlify.app` — that’s your shareable link.
4. (Optional) Sign up for a free account to edit the site name later (e.g. `valentine-for-chuzedi.netlify.app`).

**Shareable link:** Use the URL Netlify shows after you drop the folder.

---

## Option 2: GitHub Pages (good if the project is already on GitHub)

1. Push this project to a GitHub repo (e.g. `Playground` or a new repo like `valentine`).
2. On GitHub: **Settings** → **Pages**.
3. Under **Source**, choose **Deploy from a branch**.
4. Branch: **main** (or **master**), folder: **/ (root)**. Save.
5. After a minute or two, the site will be at:
   - `https://<your-username>.github.io/Playground/`  
   (if the repo name is `Playground`)

**Shareable link:**  
`https://<your-username>.github.io/Playground/`

---

## Option 3: Vercel

1. Go to **https://vercel.com** and sign in (e.g. with GitHub).
2. **Add New** → **Project** → import your repo or drag the **Playground** folder.
3. Deploy. You’ll get a URL like `https://playground-xxx.vercel.app`.

**Shareable link:** Use the URL Vercel gives you after deploy.

---

## After you have the link

- Open it on your phone and on a PC to confirm it works.
- Send the link to your girlfriend — it will work on her phone or computer.
- The page is responsive and uses touch on mobile (including the runaway “No” button).
